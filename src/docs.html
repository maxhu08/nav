rules.urls

Use this field to define URL-based rules. Each rule starts with a website regex pattern.

* <regex-pattern>

Starts a new rule block. The `* ` prefix is required before every website regex pattern.


+ <action1> <action2> ...

Blocks all actions except the listed action names for URLs matching the regex pattern above.

- <action1> <action2> ...

Allows all actions except the listed action names for URLs matching the regex pattern above.

You can list any number of action names on a `+` or `-` line.

A `+` or `-` line must go on the next consecutive line immediately after its `* <regex-pattern>` line.

If a rule block contains only `* <regex-pattern>`, all actions are blocked for matching URLs.

Separate action names with spaces.

Lines starting with # are ignored.


Example:

* ^https?://(www\.)?github\.com/.*$

+ toggle-hints-current-tab yank-current-tab-url

This blocks every action on GitHub pages except `toggle-hints-current-tab` and `yank-current-tab-url`.

hotkeys.hints.charset

Controls which letters are used to generate hint labels.

Letters earlier in the charset are prioritized first when labels are assigned.

Use unique letters. Invalid characters and duplicates are ignored.


Example:

sadfjklewcmpgh

This favors `s`, then `a`, then `d`, and so on when generating hint labels.


hotkeys.mappings

Each line maps one key sequence to one action name.

<key-sequence> <action-name>

Put one mapping on each line.

Modifier keys are specified as `<c-x>`, `<m-x>`, and `<a-x>` for ctrl+x, meta+x, and alt+x. Combined modifiers are written like `<c-a-x>` or `<c-m-x>`. Use `<space>` for the space key.

Lines starting with # are ignored.

If an action name is invalid, it is highlighted in red in the editor.


Example:

j scroll-down

gg scroll-to-top

<space> toggle-hints-current-tab

<c-y> yank-current-tab-url

yy yank-current-tab-url

This maps `j` to scroll down, `gg` to jump to the top of the page, `<space>` to toggle hints in the current tab, `<c-y>` to ctrl+y, and `yy` to copy the current tab URL.
