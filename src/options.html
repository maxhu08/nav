rules

rules.urls

* <pattern>

Starts a new rule block. The `* ` prefix is required before every pattern.


+ <action1> <action2>

Blocks all actions except the listed actions for URLs that match the pattern above.

- <action1> <action2>

Allows all actions except the listed actions for URLs that match the pattern above.

* <pattern>

If nothing comes after the pattern, all actions are blocked for matching URLs.

Separate actions with spaces.

Lines starting with # are ignored.


Example:

* https://github.com/*

+ show-hints-current-tab yank-current-tab-url

Blocks every action on GitHub pages except show-hints-current-tab and yank-current-tab-url.
